glanceproxy
===========

A multi-glance front end that supports inline image conversion and glance syncing.

Requirements
============

*) python 3

*) pyyaml

*) bottle

*) requests

*) virtualenv

*) pip/easy_install (of course)

Setup
=====

cd /path/to/your/dev/dir

virtualenv -p python3 glanceproxy

cd glanceproxy

source bin/activate

pip install pyyaml requests bottle
